# 第四周 - 快速开始指南

## 🚀 5分钟本地部署

### Windows用户

1. **双击运行部署脚本**
   ```
   双击 Week4/代码/local_deploy.bat
   ```

2. **按照提示操作**
   - 脚本会自动检查环境
   - 配置MySQL数据库
   - 初始化Django项目

3. **启动服务器**
   ```
   双击 Week4/代码/start_server.bat
   ```

4. **访问系统**
   - 本地：http://127.0.0.1:8000
   - 局域网：http://[您的IP]:8000

### Linux/Mac用户

1. **运行部署脚本**
   ```bash
   cd Week4/代码
   chmod +x local_deploy.sh start_server.sh
   ./local_deploy.sh
   ```

2. **启动服务器**
   ```bash
   ./start_server.sh
   ```

3. **访问系统**
   - 本地：http://127.0.0.1:8000
   - 局域网：http://[您的IP]:8000

## 📋 系统要求

- ✅ Python 3.9+
- ✅ MySQL 8.0+
- ✅ 4GB+ 内存

## ⚙️ 默认配置

- **数据库**：MySQL
- **用户**：reimbursement_user
- **密码**：password123
- **端口**：8000

## 🎯 验证成功

启动后看到：
```
Starting development server at http://127.0.0.0:8000/
```

打开浏览器访问显示的地址，看到登录页面即成功！
